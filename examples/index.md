# Examples

Ready-to-use YAML recipes for common motion graphics patterns.

## Available examples

### [Lower Third](/examples/lower-third)
Animated name plates and titles for video production.

### [Animated Background](/examples/animated-background)
Moving backgrounds with keyframes and effects.

### [Text Animations](/examples/text-animations)
Styled text with animated transforms.

### [Motion Templates](/examples/motion-templates)
Creating .mogrt-ready templates with Essential Graphics.

## How to use

1. Copy the YAML from any example
2. Paste it into the Compdown panel
3. Click **Create**
4. Modify as needed

## Tips

- Start with an example close to what you need
- Modify values to match your project settings
- Combine elements from different examples
- Use export to understand existing AE compositions
